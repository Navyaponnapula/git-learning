# Sales Dashboard Project

## Objective
To analyze sales data and build an interactive dashboard to understand business performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Features
- Sales trend analysis
- Region-wise performance
- Top product identification
- Monthly revenue insights

## Insights
- Identified best selling products
- Found peak sales months
- Compared regional performance
